# Zln Garage Landing

Одностраничный лендинг для автосервиса Zln Garage с HTML, CSS, JavaScript и PHP-обработкой заявок.

## Структура

```text
zln-garage-landing/
  assets/
    css/style.css
    js/main.js
    img/README.txt
  backend/
    config.php
    send.php
  index.html
  robots.txt
  sitemap.xml
  README.md
```

## Как запустить локально

Нужен PHP 7.4+.

```bash
cd zln-garage-landing
php -S localhost:8000
```

Откройте в браузере:

```text
http://localhost:8000/
```

## Где поменять контакты

Основные данные на странице находятся в `index.html`:

- телефон: `+7 (925) 053-88-33`;
- адрес: `Железнодорожная аллея, 33А, микрорайон Первомайский, Королёв`;
- ссылка на Яндекс.Карты: `https://yandex.com/maps/-/CPHnVL5-`;
- график: `ежедневно 10:00–22:00`.

Настройки отправки заявок находятся в `backend/config.php`:

- `email_enabled` — включить или выключить отправку на email;
- `email_to` — адрес получателя заявок;
- `email_from` — адрес отправителя на вашем домене;
- `telegram_enabled` — включить или выключить Telegram;
- `telegram_bot_token` — токен бота Telegram;
- `telegram_chat_id` — ID чата для заявок;
- `site_url` — реальный домен сайта.

Файл `backend/config.php` должен храниться только на сервере. Не публикуйте токен Telegram во frontend-коде.

## Как подключить к хостингу

1. Загрузите содержимое папки `zln-garage-landing` в корень сайта на PHP-хостинге.
2. Убедитесь, что `backend/send.php` доступен по пути `/backend/send.php`.
3. В `backend/config.php` укажите реальные email и Telegram-настройки.
4. При переносе с GitHub Pages на другой домен замените `https://rombensonn.github.io/zln-garage-landing/` в `index.html`, `robots.txt`, `sitemap.xml` и `backend/config.php` на реальный домен.
5. Проверьте права на папку `backend`, чтобы PHP мог создавать `leads.log` и `rate-limit.json`.

## Как проверить отправку форм

1. Запустите сайт через PHP-сервер.
2. Откройте страницу, подождите не менее 3 секунд.
3. Заполните телефон и услугу.
4. Отправьте форму.
5. Проверьте `backend/leads.log`.
6. Если включены email или Telegram, проверьте доставку заявки.

Если email на локальном компьютере не настроен, функция `mail()` может не отправлять письма. На хостинге обычно требуется корректный `email_from` на домене сайта.

## GitHub Pages

GitHub Pages публикует статическую часть сайта и не исполняет PHP. Папка `backend` сохранена в репозитории для переноса на обычный PHP-хостинг. На GitHub Pages кнопки звонка и маршрута работают, а формы требуют PHP-хостинг или отдельный серверный endpoint.

## Какие фото заменить

В проект добавлены сгенерированные тематические изображения:

- `assets/img/garage-hero.png`;
- `assets/img/tire-service.png`;
- `assets/img/engine-service.png`.

Перед боевым запуском их желательно заменить на реальные фото сервиса:

1. фасад / въезд / ориентир;
2. рабочий бокс;
3. процесс шиномонтажа;
4. автомобиль в работе;
5. фото вывески или территории;
6. схема подъезда, если есть.

После добавления фото обновите hero-фон в `assets/css/style.css` и при необходимости добавьте изображения в HTML с корректными alt-текстами.

## Антиспам

В формах и backend реализованы:

- honeypot-поле;
- проверка минимального времени заполнения формы;
- ограничение повторной отправки по IP;
- очистка HTML-тегов;
- ограничение длины полей;
- JSON-ответы без раскрытия технических ошибок пользователю.
